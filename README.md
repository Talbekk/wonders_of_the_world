# 7 Wonders of the World Educational App

7 Wonders an educational app designed for the BBC that allows users to travel around the globe learning about the wonders of the world along the way.

## Installation

Using the [node package manager](https://www.npmjs.com/get-npm), follow the instructions below to get the application up and running:

## Server

Run these instructions within the server directory:

```bash
cd server
npm install
```

Seed the database:

```bash
npm run seeds
```
Run express:

```bash
npm run server:dev
```

## Client

Run these instructions within the client directory:

```bash
cd client
npm install
npm run serve
```
